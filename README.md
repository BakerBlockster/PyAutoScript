# PyAutoScript
A basic Python program that allows for simple macros to be made and played using a DirectX library to support almost all applications and games.

This program uses Pythons built in Tkinter library to create an extremely basic and simple UI, which allows the user to record user inputs and then play them back later.

The program currently supports:
- Saving, Loading, and Deleting Macros
- Binding keybinds to Play, Stop Playing, Record, and Stop Recording
- Allows for multiple actions to have the same keybind, for e.g. Start and Stop Playing: (=) key
- Customisable safety settings to prevent accedental actions

Sadly because of the application being publically small, Windows incorrectly identifies it as possible malware since there is not signiture, so on first loading the program you need to manually tell Windows Defender to run the app anyways.
